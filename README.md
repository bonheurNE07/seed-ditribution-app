# 🌱 Seed Distribution App

A professional application for environmental organizations to register farmers, extract ID data with OCR, and distribute seeds efficiently.

## 🌐 Tech Stack
- Django + Django REST Framework
- PostgreSQL
- Google Vision API (OCR)
- Raspberry Pi for deployment
- React + Tailwind CSS (frontend - done)

## 📦 Structure
/backend → Django app
/docs → Technical documentation
/scripts → Shell & automation
